# 寻商问道 - 设计方案

## 品牌定位
**寻商问道** — AI驱动的双向智能预约平台，连接顾客与商家，实现精准匹配与智能预约。

## 色彩体系
- **主色（Primary）**: `#1A6BFF` — 科技蓝，传递智能、可信赖感
- **辅色（Secondary）**: `#FF6B35` — 活力橙，传递活跃、商业感
- **背景（Background）**: `#F8FAFF` (浅) / `#0D1117` (深)
- **卡片（Surface）**: `#FFFFFF` (浅) / `#161B22` (深)
- **文字（Foreground）**: `#0D1117` (浅) / `#F0F6FF` (深)
- **次要文字（Muted）**: `#6B7280` (浅) / `#8B949E` (深)
- **边框（Border）**: `#E5E7EB` (浅) / `#21262D` (深)
- **成功（Success）**: `#10B981`
- **警告（Warning）**: `#F59E0B`
- **错误（Error）**: `#EF4444`

## 屏幕列表

### 1. 首页 (Home)
- 顶部：欢迎语 + 用户身份切换（顾客/商家）
- 搜索框：快速搜索商家或服务
- AI助手入口卡片（醒目）
- 功能快捷入口：找商家、发布需求、我的预约、通话记录
- 近期推荐商家/顾客列表
- 底部Tab导航

### 2. AI对话 (AI Chat)
- 顶部：AI助手头像 + 名称"道道"
- 聊天气泡界面（仿ChatGPT风格）
- 支持：文字输入、语音输入
- AI能力：联网搜索、商家推荐、自动预约、模拟电话
- 底部：输入框 + 发送 + 语音按钮
- 通话状态卡片（AI正在拨打电话时显示）

### 3. 搜索/发现 (Discover)
- 搜索框（支持分类筛选）
- 分类标签：餐饮、美容、教育、医疗、家政、法律、金融...
- 商家/顾客卡片列表（头像、名称、评分、距离、简介）
- 筛选：距离、评分、价格区间、可预约时间
- 地图视图切换

### 4. 预约管理 (Appointments)
- 分类Tab：待确认、进行中、已完成、已取消
- 预约卡片：商家/顾客信息、时间、状态、操作按钮
- 预约详情页：完整信息 + 联系方式 + 取消/修改
- AI通话记录：通话时间、内容摘要

### 5. 个人中心 (Profile)
- 用户头像 + 姓名 + 身份标签
- 身份切换：顾客模式 / 商家模式
- 商家信息管理（商家模式下）
- 设置：通知、隐私、关于

## 关键用户流程

### 顾客找商家流程
1. 首页 → 点击"找商家"或AI对话
2. AI对话：描述需求 → AI联网搜索 → 推荐商家列表
3. 选择商家 → 查看详情 → 确认预约时间
4. AI模拟拨打电话确认 → 预约成功
5. 预约管理页查看状态

### 商家找顾客流程
1. 切换商家模式 → 首页 → "发布需求"
2. AI对话：描述目标客户 → AI匹配潜在顾客
3. 选择顾客 → AI代为联系
4. 顾客确认 → 预约成功

## 视觉风格
- 整体风格：现代简约，类ChatGPT + 中国商业App风格
- 圆角：卡片16px，按钮12px，头像50%
- 阴影：轻微投影，增加层次感
- 字体：系统字体，标题粗体，正文常规
- 图标：线性图标，简洁现代
- 动效：流畅过渡，AI思考时的打字动画
